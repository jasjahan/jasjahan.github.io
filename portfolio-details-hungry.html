Project detail

  • A Restaurant web application for restaurant search, food order, and pickup.
  • Due to the Covid-19 situation, this full-stack development process has been done by communication via Zoom. Team collaboration has been done by Github and Trello.
  • The process included stack tools selection, use case analysis, high-level requirements and architecture, design mockups, database design, testing.
  • Features included searching, login, log up, order online, pickup, delivery, review, customer side, admin side, owner side, driver side cases.
  • Used Python, Flask as the server-side language, UML to view the overall design, MySQL for the database, and Amazon AWS for running the application instance to the back end.
